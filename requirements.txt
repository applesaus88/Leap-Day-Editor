# --- Platform notes -------------------------------------------------------
# Windows: pywebview uses the Edge WebView2 runtime (pre-installed on Win 11;
#          Win 10 may need the free "Evergreen" WebView2 runtime from Microsoft).
# Linux:   pywebview needs a GTK backend — install system packages
#          `python3-gi gir1.2-webkit2-4.1` (Debian/Ubuntu) then `pip install pycairo PyGObject`.
# macOS:   uses the built-in WebKit — no extra runtime needed.
#
# Building/installing an APK additionally needs Java 17+ (signing) and, to
# install to a device, `adb` with an arm64 Android emulator.
