# Example mods

These `.ldmod` files are **original demo levels authored in the editor** — they
contain only user-created level layouts (tiles, enemies, difficulty rating), no
game code, art, or audio. Anyone can open one in the studio, apply it to their
own copy of the game, and play it.

## Try one
1. Launch the studio: `python3 studio/app.py`
2. **Load Game (.xapk)** — supply your own copy of the game.
3. **Open .ldmod** → pick a file from this folder.
4. **Build APK** (or **Build + Install** with an emulator connected).

## Make your own
Author a level in the editor, then **Save .ldmod** and drop it here. Because a
`.ldmod` is only your authored content, these are safe to share and commit.
